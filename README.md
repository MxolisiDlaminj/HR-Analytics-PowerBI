# 📊 HR Analytics Dashboard – Power BI

## Overview

This project is an interactive **HR Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard transforms raw Human Resources data into meaningful insights that support workforce analysis and informed business decision-making.

The project demonstrates my ability to clean, model, analyze, and visualize data using Power BI.

---

## Objectives

* Analyze employee demographics and workforce distribution.
* Monitor key HR performance indicators (KPIs).
* Identify trends in employee attrition.
* Provide interactive reports for HR decision-making.
* Present insights through clear and professional visualizations.
